SkinnyCoder

SkinnyCoder is a lean coding assistant workflow focused on fast, direct software changes. It reads the project, plans only what is needed, edits files with minimal ceremony, and reports the result without turning a small task into a large process.

Why it is skinny

It is skinny because it keeps the interface and workflow compact. No heavy dashboards, no noisy rituals, and no bloated output. SkinnyCoder favors clear actions, small files, direct commands, and practical progress.

Slash commands

  • /plan - outline the next steps before changing code.
  • /read - inspect files and understand the current project.
  • /edit - make focused file changes.
  • /run - execute a command or check the result.
  • /ship - summarize what changed and what was verified.
